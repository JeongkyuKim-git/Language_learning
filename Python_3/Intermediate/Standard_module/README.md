*math (수학 모듈)*
-------------  
>   math는 기본적인 수학 모듈입니다. 여러 수학적인 함수를 제공해 줍니다.  
```python
import math

# 코사인 함수 (모든 삼각함수는 라디안을 사용합니다)
print(math.cos(0))

# 로그 함수
print(math.log10(100))
```  
> 1.0  
> 2.0  

*random (난수 모듈)*
-------------  
> random 모듈은 랜덤 한 숫자를 생성하기 위한 다양한 함수들을 제공해 줍니다.  
```python
import random

# 랜덤한 정수 1 <= N <= 20 
print(random.randint(1, 20))

# 랜덤한 소수 0 <= x <= 1
print(random.uniform(0, 1))
```
