*Repository, Commit*
-------------
> 커밋 (commit)  
> 프로젝트 디렉토리의 특정 모습을 하나의 버전으로 남기는 행위 & 결과물  

> 레포지토리 (repository)  
> 커밋이 저장되는 곳  

*Repository*
-------------
> 레포지토리 (폴더 생성 및 git에 연동)  
> mkdir [폴더 이름] --> cd [폴더 이름] --> git init
